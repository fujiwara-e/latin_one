## Setup
1. 本プロジェクト を clone する
```bash
git clone git@github.com:hosokawa-kenshin/latin_one.git
```
2. 依存関係をインストールする
```bash
cd latin_one
flutter pub get
```
3. アプリを実行する
```bash
flutter run
```

## Features
- データベース (Firebase Firestore)
- プッシュ通知 (Firebase Cloud Messaging)
- コーヒー豆の注文
